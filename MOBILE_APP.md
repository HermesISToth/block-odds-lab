# Mobile App Plan

## Current Version

Block Odds Lab is now an installable Progressive Web App.

Live URL:

https://hermesistoth.github.io/block-odds-lab/

## Install On iPhone

1. Open the live URL in Safari.
2. Tap the Share button.
3. Tap `Add to Home Screen`.
4. Confirm the name `Block Odds`.

## Install On Android

1. Open the live URL in Chrome.
2. Tap the browser menu.
3. Tap `Install app` or `Add to Home screen`.
4. Confirm install.

## Later Native App Path

When funding and accounts are ready:

- Android: wrap the PWA with Trusted Web Activity or Capacitor and publish through Google Play.
- iPhone: wrap with Capacitor and publish through the Apple App Store.

Store releases require developer accounts, store assets, privacy disclosures, and review time. The PWA lets us validate demand first without spending money.
