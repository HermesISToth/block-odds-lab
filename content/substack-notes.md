# Block Odds Lab Substack Notes

Use these as short Substack Notes. Each one should include a clickable BOL link.

1. A NerdMiner can technically find a Bitcoin block. That sentence is true and still wildly misleading without the odds. Run the math before treating "possible" like "probable." [NerdMiner Bitcoin odds](https://blockoddslabs.com/nerdminer-bitcoin-odds.html)

2. Expected time is not a countdown. A solo miner is not slowly earning its way toward a block. Every hash is a fresh chance against current difficulty. [Block odds calculator](https://blockoddslabs.com/block-odds-calculator.html)

3. Bitaxe vs NerdMiner is not just a culture choice. It is a hashrate gap, a power gap, a heat gap, and an odds gap. [Bitaxe odds calculator](https://blockoddslabs.com/bitaxe-odds-calculator.html)

4. The healthiest version of lottery mining is knowing the odds and still enjoying the experiment. The unhealthy version is pretending the math is better than it is. [Bitcoin solo mining calculator](https://blockoddslabs.com/bitcoin-solo-mining-calculator.html)

5. Before buying another hobby miner, ask: would I still be happy owning this if it never finds a block? If yes, it might be a hobby. If no, run the odds again. [Run the odds](https://blockoddslabs.com/block-odds-calculator.html)

6. Some low-cost lottery miner upgrades do not improve luck. They improve stability: better power, better cooling, fewer rejected shares, fewer flaky cables. [Low-cost lottery miner upgrades](https://blockoddslabs.com/articles/low-cost-upgrades.html)

7. Solo mining math is emotionally hard because the prize is easy to imagine and the base rate is not. That is exactly why calculators help. [Block odds calculator](https://blockoddslabs.com/block-odds-calculator.html)

8. If your miner has a great screenshot hashrate but crashes overnight, the screenshot is not the setup. Stability beats vanity hashrate. [Bitaxe Gamma tuning guide](https://blockoddslabs.com/articles/bitaxe-gamma-tuning.html)

9. The pages that need attention are not always the pages that feel exciting. If Google has not found the calculator route yet, the job is boring but important: link it, share it where it helps, and make the next click obvious. [Block odds calculator](https://blockoddslabs.com/block-odds-calculator.html)

10. Bitcoin difficulty makes old lottery-mining screenshots expire. If a Bitaxe or NerdMiner post is using last week's odds, rerun the math before arguing about the setup. [Difficulty tracker](https://blockoddslabs.com/bitcoin-mining-difficulty-tracker.html)

11. A second small miner does not double your certainty. It doubles hashrate, which still has to be measured against current Bitcoin difficulty, watts, and time horizon. [Bitaxe odds calculator](https://blockoddslabs.com/bitaxe-odds-calculator.html)

12. The honest sales pitch for lottery mining is not "you will hit a block." It is "you will learn Bitcoin mining by watching real hardware fight impossible odds." [Starter kit](https://blockoddslabs.com/lottery-mining-starter-kit.html)
