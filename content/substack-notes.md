# Block Odds Lab Substack Notes

Use these as short Substack Notes after the publication exists. Each one should include one BOL link.

1. A NerdMiner can technically find a Bitcoin block. That sentence is true and still wildly misleading without the odds. Run the math before treating "possible" like "probable." https://blockoddslabs.com/nerdminer-bitcoin-odds.html

2. Expected time is not a countdown. A solo miner is not slowly earning its way toward a block. Every hash is a fresh chance against current difficulty. https://blockoddslabs.com/block-odds-calculator.html

3. Bitaxe vs NerdMiner is not just a culture choice. It is a hashrate gap, a power gap, a heat gap, and an odds gap. https://blockoddslabs.com/bitaxe-odds-calculator.html

4. The healthiest version of lottery mining is knowing the odds and still enjoying the experiment. The unhealthy version is pretending the math is better than it is. https://blockoddslabs.com/bitcoin-solo-mining-calculator.html

5. Before buying another hobby miner, ask: would I still be happy owning this if it never finds a block? If yes, it might be a hobby. If no, run the odds again. https://blockoddslabs.com/block-odds-calculator.html

6. Some low-cost lottery miner upgrades do not improve luck. They improve stability: better power, better cooling, fewer rejected shares, fewer flaky cables. https://blockoddslabs.com/articles/low-cost-upgrades.html

7. Solo mining math is emotionally hard because the prize is easy to imagine and the base rate is not. That is exactly why calculators help. https://blockoddslabs.com/block-odds-calculator.html

8. If your miner has a great screenshot hashrate but crashes overnight, the screenshot is not the setup. Stability beats vanity hashrate. https://blockoddslabs.com/articles/bitaxe-gamma-tuning.html
