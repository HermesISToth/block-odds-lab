# Paid Product Spec: Lottery Miner Tuning Dashboard

## Product Promise

Help lottery miner owners understand whether an overclock is actually stable, what it costs to run, and how much it changes block odds.

## Free Lead Magnet

`site/downloads/bitaxe-tuning-log.csv`

Purpose:
- Give immediate value.
- Collect future email interest once a list provider exists.
- Teach the exact data model for the paid dashboard.

## Paid Version

Working name: Block Odds Pro Log

Possible price:
- Launch: $5
- Mature version: $9-$15

Features:
- Tuning log with frequency, voltage, temps, fan, power, errors, uptime.
- Auto odds calculator per setting.
- Electricity cost estimate.
- Stability score.
- Best-setting recommendation based on overnight run.
- Export/share summary for Reddit/Discord.
